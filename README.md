## 🚀 **Projeto em Destaque — Sistema de Cadastro e Login Seguro em PHP**

📂 [**Repositório Completo no GitHub**](https://github.com/LeoTerryMaster/Cadastrolandia/tree/raiz)  
🎥 [**Assista à Explicação no YouTube**](https://www.youtube.com/watch?v=cO9BOIWM9ME)

> 🔐 Projeto completo de autenticação segura com **PHP 8+, PDO, CSRF e mensagens dinâmicas**,  
> desenvolvido com foco em **boas práticas, segurança e arquitetura limpa**.

---

### 💡 **Destaques do Projeto**
- ✍️ Validação completa de campos (nome, email, senha)  
- 🔐 Proteção total com **CSRF Tokens**  
- ⚙️ Lógica organizada com `switch/case`  
- 💬 Mensagens de sucesso e erro dinâmicas  
- 🧱 **PDO com prepared statements** contra SQL Injection  
- 🧭 Estrutura modular (Helpers, DAO, Controllers e Views)  
- 🐞 Página de debug completa (`logado.php`)  
- 📱 Interface moderna com **Bootstrap 5**  

🧠 **Stack:** `PHP 8.2` · `PDO` · `CSRF Protection` · `HTML5` · `CSS3` · `Bootstrap`  
💾 **Banco de Dados:** MySQL (leve, rápido e seguro)  

> 🎯 *Perfeito para quem quer entender na prática como construir um sistema de login seguro do zero.*
